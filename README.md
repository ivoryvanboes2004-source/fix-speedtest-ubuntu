# Fix Speedtest Ubuntu (noble → jammy)

Workaround om Ookla `speedtest` te installeren op Ubuntu 24.04 (noble) door de repo naar `jammy` te switchen.

## Run
```bash
curl -fsSL https://raw.githubusercontent.com/ivoryvanboes2004-source/fix-speedtest-ubuntu/main/fix-speedtest-ubuntu.sh -o fix-speedtest-ubuntu.sh
chmod +x fix-speedtest-ubuntu.sh
sudo ./fix-speedtest-ubuntu.sh
